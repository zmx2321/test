## 使用

博客采用 vuepress 作为主框架。

```sh
# clone with Git Bash
git clone https://github.com/zmx2321/blog.git

# change directory
cd blog

# install dependencies
yarn

# serve with hot reload at localhost:8080
yarn docs:dev

# build for production with minification
yarn build
```
